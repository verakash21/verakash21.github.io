Master's Thesis

Histopathology H&E images classification using unsupervised learning.

Using features extracted from H&E images of prostate cancer tissues to classify cancer. The system developed can extract different features including color, shape and size of cells and apply zernike moments, which are orthogonal and rotation independent. These features are analyzed using artificial neural network that is trained using unsupervised learning to do dimensionality reduction.
